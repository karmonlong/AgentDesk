# 🚀 智能体指挥中心 - 使用指南

## 🎯 什么是指挥中心？

**智能体指挥中心（Agent Command Center）** 是一个赛博朋克风格的可视化控制台，让你像指挥官一样管理和调度智能体团队。相比传统聊天界面，它提供：

- 🎨 **科技感设计**：赛博朋克风格，炫酷的视觉效果
- 📊 **工作流可视化**：实时看到智能体的工作流程
- 🔍 **状态监控**：监控每个智能体的状态和负载
- ⚡ **任务控制台**：像黑客一样输入命令
- 🌐 **粒子特效**：动态背景和连接线动画

## 🎨 界面特性

### 1. 顶部状态栏
```
╔═════════════════════════════════════════════════════╗
║ 🧠 AGENT COMMAND CENTER    [●] 系统在线 3/6活跃 ║
╚═════════════════════════════════════════════════════╝
```

**显示信息**：
- 系统在线状态（绿色呼吸灯）
- 活跃智能体数量（实时更新）
- 已完成任务数
- 系统时间

### 2. 左侧 - 智能体面板

每个智能体显示为一张**发光卡片**：

```
┌────────────────────────────┐
│  📊              [WORKING]  │ ← 状态徽章
│  文档分析师                 │
│  信息提取与分析专家         │
│  任务: 5      负载: 0%     │ ← 统计信息
└────────────────────────────┘
    ↑ 鼠标悬停时发光
```

**状态类型**：
- 🟢 **IDLE** - 空闲状态（灰色）
- 🟡 **WORKING** - 工作中（绿色发光+脉冲动画）
- 🔴 **ERROR** - 错误状态（红色）

**交互**：
- 悬停时卡片发光并向右移动
- 工作时持续脉冲动画
- 点击可查看详情（开发中）

### 3. 中间 - 工作流可视化 (Canvas)

**圆形节点布局**：
```
        📊 文档分析师
           ╱         ╲
         ╱             ╲
      ✍️               📈
    内容创作者        数据专家
         ╲             ╱
           ╲         ╱
        ✅ 校对编辑
```

**视觉效果**：
- ⚪ 智能体显示为圆形节点（带emoji）
- 💫 工作中的节点有外圈光晕
- ⚡ 任务传递时显示流动的连接线
- ✨ 粒子沿着连接线移动

**控制按钮**：
- 🧹 **清除** - 清空工作流历史
- 💾 **导出** - 导出工作流图（开发中）
- 🎨 **自动布局** - 重新排列节点

### 4. 右侧 - 任务控制台

**快捷操作**（一键使用模板）：
```
┌──────────┬──────────┐
│ 📊 文档  │ ✍️ 内容  │
│   分析   │   创作   │
├──────────┼──────────┤
│ 📈 数据  │ 🌐 翻译  │
│   分析   │   文档   │
└──────────┴──────────┘
```

**命令输入框**（终端风格）：
```
> 输入任务指令...
> 可使用 @智能体名称 指定执行者
> 示例：@文档分析师 分析季度报告
```

**执行按钮**（渐变发光）：
```
╔══════════════════════════╗
║  ▶ 执行任务              ║
╚══════════════════════════╝
```

**控制台日志**（实时滚动）：
```
[00:12:45] USER: 任务 #1: @文档分析师 分析报告
[00:12:46] ROUTER: 根据@提及选择了文档分析师
[00:12:46] 文档分析师: 开始处理任务...
[00:12:48] 文档分析师: 任务完成
```

## 🎮 使用方法

### 方式1：快捷操作（最简单）

1. 点击右侧**快捷操作**按钮之一
2. 系统自动填充命令模板
3. 点击"**执行任务**"按钮
4. 观察工作流可视化区域的动画

### 方式2：直接输入命令

```bash
# 指定智能体
> @文档分析师 分析这份季度报告

# 自动路由（系统自动选择）
> 帮我写一封感谢信

# 复杂任务
> @数据专家 分析销售数据并生成趋势图
```

### 方式3：键盘快捷键

- `Ctrl/Cmd + Enter` - 执行任务
- `Esc` - 清空输入
- 鼠标滚轮 - 滚动日志

## 💡 设计理念

### 1. 赛博朋克美学

灵感来源：
- 《黑客帝国》的绿色代码雨
- 科幻电影中的全息界面
- 游戏《赛博朋克2077》的UI

**色彩方案**：
```css
主色：#00ff88 (霓虹绿)
辅色：#00d4ff (电子蓝)
强调：#ff00ff (紫红)
背景：#0a0e27 (深空蓝)
```

### 2. 信息可视化

**目标**：让抽象的"智能体协作"变得具象

- 智能体 → 发光节点
- 任务传递 → 流动连接线
- 工作状态 → 光晕和动画
- 日志 → 终端输出

### 3. 交互反馈

**每个操作都有视觉反馈**：
- 点击按钮 → 发光效果
- 执行任务 → 节点开始脉冲
- 任务完成 → 光晕消失
- 鼠标悬停 → 卡片发光

## 🔧 技术实现

### Canvas 动画系统

```javascript
// 60fps 动画循环
function animate() {
    clearCanvas();
    drawConnections();  // 绘制连接线
    drawAgentNodes();   // 绘制智能体节点
    requestAnimationFrame(animate);
}
```

### 状态管理

```javascript
agents = [
    {
        name: "文档分析师",
        status: "idle",      // idle | working | error
        position: {x, y},    // Canvas坐标
        tasksCompleted: 0    // 已完成任务数
    },
    // ...
]
```

### 连接线动画

```javascript
connection = {
    from: agentIndex1,
    to: agentIndex2,
    progress: 0 → 1,  // 动画进度
    duration: 1500ms
}
```

## 🌟 亮点总结

1. **可视化工作流**

   - 能看到任务如何在智能体间流转

2. **实时状态监控**
   - 知道哪个智能体在工作
   - 看到工作负载

3. **科技感设计**
   - 不像"聊天工具"
   - 像"指挥系统"

4. **系统级视角**
   - 不是"我和AI对话"
   - 而是"我指挥AI团队"
   - 提升价值感知
