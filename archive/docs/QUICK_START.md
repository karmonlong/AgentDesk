# 🚀 快速开始 - 三种界面使用指南

## 📋 选择合适的界面

### 场景1：我有文档文件要上传

**推荐界面**：对话界面 (`/chat`) 或 传统界面 (`/`)

```bash
# 访问对话界面
http://localhost:8000/chat

# 或访问传统界面
http://localhost:8000/
```

### 场景2：我想粘贴文本内容处理

**推荐界面**：指挥中心 (`/command`) 或 对话界面 (`/chat`)

### 场景3：我想演示/监控系统

**推荐界面**：指挥中心 (`/command`)

---

## 1️⃣ 指挥中心使用教程 (当前界面)

### 📝 文本处理（无需上传文件）

**步骤**：

1. **使用快捷操作**
   - 点击右侧的快捷按钮（如"📊 文档分析"）
   - 自动填充命令模板

2. **在输入框中输入/粘贴内容**
   ```
   @文档分析师 分析以下季度报告：
   
   2024年Q3业绩报告
   总营收：1250万元
   同比增长：35%
   ...（粘贴你的文档内容）
   ```

3. **点击"执行任务"按钮**
   - 观察中间Canvas的动画
   - 查看智能体节点发光
   - 阅读底部日志输出

4. **查看结果**
   - 在控制台日志中查看AI的回复

### 🎯 命令格式

```bash
# 格式1：指定智能体
@智能体名称 任务描述

# 格式2：自动路由（不指定智能体）
任务描述

# 示例
@文档分析师 总结这份报告
@内容创作者 写一封感谢信
@数据专家 分析销售趋势
```

### ⚡ 快捷键

- `Ctrl/Cmd + Enter` - 执行任务
- 鼠标滚轮 - 滚动日志

## 2️⃣ 对话界面使用教程

### 访问地址
```
http://localhost:8000/chat
```

### 📤 上传文件步骤

1. **点击输入框旁的 📎 图标**
   - 位置：输入框右侧

2. **选择文件**
   - 支持：TXT, PDF, DOCX, XLSX, CSV, MD, JSON
   - 最大：50MB

3. **输入任务指令**
   ```
   @文档分析师 分析这份文档
   ```
   或
   ```
   帮我总结这份报告
   ```

4. **发送消息**
   - 点击发送按钮或按 Enter

5. **持续对话**
   - AI回复后可以继续追问
   - 系统会记住上下文

### 💬 对话技巧

```bash
# 使用 @ 提及特定智能体
@文档分析师 提取关键数据
@内容创作者 写成正式报告
@校对编辑 检查语法错误

# 不使用 @，让系统自动选择
总结这份文档
写一封邮件
分析这些数据
```

## 3️⃣ 传统界面使用教程

### 访问地址
```
http://localhost:8000/
```

### 📤 上传处理步骤

1. **选择文件**
   - 点击"选择文件"按钮
   - 或拖拽文件到页面

2. **选择操作类型**
   - 总结文档
   - 生成内容
   - 格式转换
   - 提取表格
   - 提取要点
   - 深度分析

3. **填写额外指示**（可选）
   ```
   示例：
   "重点关注财务数据"
   "生成邮件格式"
   "提取关键时间节点"
   ```

4. **点击"开始处理"**

5. **下载结果**
   - 处理完成后点击"下载结果"按钮

## 🎯 常见问题

### Q: 指挥中心能上传文件吗？

**A**: 不能。指挥中心主要用于：
- 可视化监控
- 命令式交互（粘贴文本）
- 演示展示

如需上传文件，请使用对话界面 (`/chat`)

### Q: 如何在指挥中心处理文档？

**A**: 两种方式：
1. **直接粘贴文本**到输入框
2. **先在对话界面上传**，在指挥中心监控

### Q: 三种界面可以同时使用吗？

**A**: 可以！它们共享同一个智能体系统：
```
/chat    - 上传文件并对话
/command - 监控工作流程
/        - 快速批量处理
```

### Q: 如何选择智能体？

**A**: 两种方式：
1. **手动指定**：使用 @ 提及
   ```
   @文档分析师 分析文档
   @内容创作者 写报告
   ```

2. **自动路由**：直接描述任务
   ```
   总结这份报告 → 自动选择文档分析师
   写一封邮件 → 自动选择内容创作者
   ```

### Q: 如何看到处理结果？

**界面不同，结果显示不同**：

| 界面 | 结果显示位置 |
|------|------------|
| 传统界面 | 结果预览区 + 下载按钮 |
| 对话界面 | 对话消息中 |
| 指挥中心 | 控制台日志 |
