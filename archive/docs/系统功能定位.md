## 🎯 一、系统核心定位

### 1.1 系统愿景

打造**国内首个基于多智能体协作的基金行业AI工作台**，通过自然语言交互和智能工作流编排，让AI成为基金从业人员的智能协作伙伴，全面提升日常办公、投研分析、营销合规、运营管理的效率和质量。

### 1.2 核心价值主张

#### **相比传统方式的优势**

| 维度 | 传统方式 | AI智能体平台 | 提升幅度 |
|------|----------|--------------|----------|
| **文档处理** | 手工阅读+人工总结 | AI自动提取+结构化 | 效率提升80% |
| **投研分析** | 逐份研读研报+Excel | 智能提取+自动分析 | 时间节省70% |
| **合规审核** | 人工逐字检查 | AI实时审核+自动修正 | 准确率提升90% |
| **营销内容** | 人工撰写+多轮审核 | AI生成+闭环审核 | 周期缩短60% |
| **知识管理** | 分散存储+难以复用 | 智能检索+上下文理解 | 复用率提升5倍 |

#### **核心差异化能力**

```
🤖 多智能体协作
   ├─ 7+专业智能体各司其职
   ├─ 场景化智能路由
   └─ 人机协同审核

🔄 LangGraph工作流
   ├─ 状态持久化与恢复
   ├─ 复杂任务自动编排
   └─ 条件分支与错误处理

⚖️ 基金行业深度适配
   ├─ 合规知识库内置
   ├─ 投研工作流预设
   └─ 营销审核闭环

🎨 渐进式用户体验
   ├─ 零学习成本（自然语言）
   ├─ 三种交互模式
   └─ 实时可视化反馈
```

### 1.3 目标用户

#### **主要用户群体**

1. **投研人员** (40%)
   - 基金经理、研究员
   - 需求：快速分析研报、提取数据、生成投资建议

2. **合规/风控人员** (25%)
   - 合规官、风控专员
   - 需求：营销材料审核、风险识别、法规检索

3. **市场营销人员** (20%)
   - 产品经理、渠道专员
   - 需求：文案生成、合规审核、客户材料制作

4. **运营/行政人员** (15%)
   - 运营专员、行政助理
   - 需求：文档处理、报表生成、会议纪要

### 1.4 系统边界

#### **✅ 我们做什么**

- ✅ **智能文档处理**: 总结、提取、转换、分析
- ✅ **投研工作流自动化**: 研报分析、数据提取、趋势判断
- ✅ **合规审核闭环**: 自动审核 → 修改建议 → 复审 → 通过
- ✅ **营销内容生成**: 文案撰写 + 合规审核一体化
- ✅ **多智能体协作**: 复杂任务自动分解与编排
- ✅ **自然语言交互**: 零学习成本，像和同事对话一样

#### **❌ 我们不做什么**

- ❌ **不做交易执行**: 不直接对接交易系统
- ❌ **不做投资决策**: 只提供分析参考，不做最终决策
- ❌ **不替代人工审核**: 关键环节保留人工审核节点
- ❌ **不存储敏感数据**: 不持久化客户敏感信息
- ❌ **不做客户直接服务**: B端工具，不面向C端客户

## 🏢 二、业务场景分类

### 2.1 日常办公场景

#### **核心需求**
```
员工日常办公中遇到的文档处理、沟通协作任务
```

#### **典型任务**

**📄 文档处理**
- 总结会议纪要，提取行动项
- 转换文档格式（PDF → Word → Markdown）
- 提取合同/报告关键信息
- 多文档对比分析

**✉️ 邮件撰写**
- 根据要点生成正式邮件
- 邮件语气调整（正式/友好/简洁）
- 批量邮件模板生成

**📝 报告生成**
- 月度/季度工作报告
- 项目进展汇报
- 数据分析报告

**工作流示例**：
```
Word会议纪要 
  → @文档分析师(提取要点) 
  → @内容创作者(生成邮件) 
  → @校对编辑(优化) 
  → 输出邮件草稿
```

#### **智能体分工**
- 主力：文档分析师、内容创作者、校对编辑
- 辅助：翻译专家、协调者

---

### 2.2 投研场景

#### **核心需求**
```
投资研究人员需要快速处理大量研报、财报，提取关键信息，生成投资分析
```

#### **典型任务**

**📊 研报分析**
- 快速总结券商研报核心观点
- 提取盈利预测、估值数据
- 对比多家机构观点
- 识别投资逻辑和风险点

**💰 财报数据提取**
- 从PDF财报提取关键财务指标
- 同比环比数据计算
- 生成财务指标趋势表
- 异常数据标注

**🔍 行业研究**
- 整理行业政策文件
- 提取行业数据和趋势
- 对比行业内公司表现
- 生成行业分析报告

**📈 投资建议生成**
- 基于研报和财报生成投资要点
- 风险提示整理
- 估值分析
- 投资建议草稿

**工作流示例**：
```
券商研报PDF 
  → @文档分析师(提取核心观点+数据) 
  → @数据专家(分析趋势+计算指标) 
  → @投研分析师(生成投资建议) 
  → @风控专家(识别风险点) 
  → 输出投资分析报告
```

#### **智能体分工**
- 主力：**投研分析师**（新增）、数据专家、文档分析师
- 辅助：风控专家、内容创作者

#### **数据来源**
- PDF研报（券商、第三方机构）
- Excel财报数据
- 行业政策文件
- 市场数据（需对接API）

---

### 2.3 营销场景

#### **核心需求**
```
市场营销人员需要快速生成营销文案，并确保内容合规，通过审核
```

#### **典型任务**

**✍️ 营销文案生成**
- 基金产品介绍文案
- 投资者教育内容
- 市场分析解读
- 活动宣传文案

**⚖️ 合规审核（重点功能）**
- 识别违规词汇（保本、稳赚、无风险等）
- 检查风险揭示是否充分
- 引用法规条款
- 提供修改建议

**📢 客户沟通材料**
- 产品说明书
- 客户问答FAQ
- 市场观点邮件
- 路演材料

**🔄 营销内容闭环流程**
```
主题输入
  ↓
@内容创作者(生成营销文案)
  ↓
@合规官(第一次审核)
  ├─ 通过 → 输出
  └─ 不通过 → 修改建议
       ↓
@内容创作者(根据建议修改)
  ↓
@合规官(复审)
  ├─ 通过 → 输出
  └─ 不通过 → 人工介入
```

#### **智能体分工**
- 主力：**合规官**（核心）、内容创作者
- 辅助：校对编辑、文档分析师

#### **合规知识库**
```
内置基金行业法规：
- 《基金销售管理办法》
- 《证券投资基金法》
- 《公开募集证券投资基金信息披露管理办法》
- 常见违规词汇库
- 风险提示标准模板
```
