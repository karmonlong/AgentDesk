# AKShare 数据专家使用指南

## 📊 简介

**AKShare 数据专家** 是一个专门用于查询中国资本市场数据的智能体，基于 AKShare MCP 工具提供实时股票数据查询服务。

## 🎯 核心功能

### 1. 股票信息查询
查询股票的基本信息，包括公司名称、行业、市值等。

**示例**：
```
@AKShare数据专家 查询 000001 的股票信息
@AKShare数据专家 查询平安银行的基本信息
```

### 2. 股票价格数据
获取股票的历史价格数据、K线数据。

**示例**：
```
@AKShare数据专家 查询 600036 的最近 10 条价格数据
@AKShare数据专家 查询招商银行的日线数据
```

### 3. 股票新闻查询
获取与特定股票相关的最新新闻。

**示例**：
```
@AKShare数据专家 查询 600276 的最新新闻
@AKShare数据专家 恒瑞医药有什么新闻？
```

### 4. 股票代码搜索
通过公司名称搜索对应的股票代码。

**示例**：
```
@AKShare数据专家 搜索"平安银行"相关的股票
@AKShare数据专家 查找茅台的股票代码
```

### 5. 交易日查询
获取当前时间和交易日信息。

**示例**：
```
@AKShare数据专家 查询当前时间和交易日信息
@AKShare数据专家 今天是交易日吗？
```

## 🖥️ 使用方式

### 方式一：侧边栏面板（推荐）

1. **点击侧边栏图标**：在左侧活动栏找到 📈 图标（AKShare 数据专家）
2. **选择查询类型**：从下拉菜单选择要查询的数据类型
3. **输入股票代码或关键词**：
   - 股票代码：如 `600276`、`000001`
   - 公司名称：如 `平安银行`、`招商银行`
4. **设置参数**（可选）：
   - 数据条数：默认 10 条
   - 周期：日线/周线/月线
5. **点击"查询数据"按钮**

**快速示例按钮**：
- 📊 平安银行信息
- 📰 招商银行新闻
- 🕐 交易日查询

### 方式二：聊天输入框

直接在聊天框中使用 `@AKShare数据专家` 提及智能体：

```
@AKShare数据专家 查询 600276 的股票信息
```

### 方式三：智能体市场

1. 打开智能体市场
2. 找到 "AKShare数据专家"
3. 点击订阅或直接使用

## 📝 查询建议

### ✅ 推荐做法

1. **使用股票代码查询**（准确度最高）：
   ```
   @AKShare数据专家 查询 600276 的股票信息
   ```

2. **明确指定查询类型**：
   ```
   @AKShare数据专家 查询 000001 的价格数据
   ```

3. **先搜索代码，再查询详情**：
   ```
   第一步：@AKShare数据专家 搜索"恒瑞医药"
   第二步：@AKShare数据专家 查询 600276 的详细信息
   ```

### ⚠️ 注意事项

1. **公司名称搜索限制**：
   - 搜索功能可能受数据源限制
   - 部分公司名称可能搜索不到
   - 建议直接使用股票代码

2. **数据时效性**：
   - 数据来自公开数据源
   - 可能存在延迟
   - 非交易时间数据不会更新

3. **网络依赖**：
   - 需要连接到 AKShare MCP 服务
   - 网络不稳定时可能超时
   - 超时后会降级到 LLM 知识库

## 🔧 技术特性

### 智能降级机制

当 MCP 工具不可用时，系统会自动降级到 LLM 的训练知识：

```
⚠️ 无法使用实时数据工具

我将基于训练知识为您提供参考信息...
```

### 超时保护

- MCP 工具调用设置 10 秒超时
- 超时后自动降级，不会无限等待
- 提供友好的错误提示

### 参数自动格式化

系统会自动：
- 规范化股票代码格式
- 推断市场类型（A股/港股/美股）
- 验证参数有效性

## ❓ 常见问题

### Q1: 为什么搜索公司名称找不到？
**A**: 搜索功能受数据源限制，建议直接使用股票代码查询。

### Q2: 查询一直转圈怎么办？
**A**: 可能是网络超时，系统会在 10 秒后自动降级到 LLM 知识库。

### Q3: 如何知道股票代码？
**A**: 使用搜索功能：`@AKShare数据专家 搜索"公司名称"`

### Q4: 数据是实时的吗？
**A**: 数据来自公开数据源，可能存在延迟，非交易时间不会更新。
