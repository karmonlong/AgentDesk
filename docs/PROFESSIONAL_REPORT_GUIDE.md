# 📊 专业分析报告创建指南

## 🎯 概述

本指南将帮助您使用 AgentDesk 创建包含**文字、HTML图表、Mermaid图表、Nano Banana图片**的专业分析报告。

---

## 🚀 方式一：分步骤创建（推荐）

### 步骤 1：创建文字内容框架

**指令示例**：
```
@内容创作者 请为我创建一份关于"2024年基金市场分析报告"的框架，包括：
1. 执行摘要
2. 市场概况
3. 数据分析
4. 趋势预测
5. 投资建议

要求：专业、结构清晰、适合加入图表和图片
```

### 步骤 2：生成 Mermaid 流程图/架构图

**指令示例**：
```
@绘画智能体 根据以下内容生成一个Mermaid流程图：
展示基金投资决策流程：市场分析 -> 风险评估 -> 资产配置 -> 执行交易 -> 监控调整

工具：只使用 Mermaid
```

### 步骤 3：生成 HTML 数据图表

**指令示例**：
```
@数据可视化专家 生成一个柱状图，展示2024年各季度基金收益率：
Q1: 5.2%, Q2: 7.8%, Q3: 4.5%, Q4: 6.3%
```

### 步骤 4：生成 Nano Banana 配图

**指令示例**：
```
@绘画智能体 生成一张展示"基金投资组合"的配图，风格：专业、商务、现代

工具：只使用 Nano Banana
```

### 步骤 5：整合所有内容

**指令示例**：
```
@内容创作者 请将以下内容整合成一份完整的专业报告：

【文字框架】
[粘贴步骤1的内容]

【Mermaid流程图】
[粘贴步骤2的代码]

【HTML图表】
[粘贴步骤3的代码]

【配图说明】
[粘贴步骤4的图片]

要求：
1. 保持专业格式
2. 在合适位置插入图表和图片
3. 使用Markdown格式
4. 确保内容连贯流畅
```

## 🎯 方式二：一次性指令（智能体协作）

### 完整指令模板

```
@内容创作者 @数据可视化专家 @绘画智能体

请协作创建一份关于"[主题]"的专业分析报告，要求：

**报告结构**：
1. 执行摘要（文字）
2. 市场概况（文字 + HTML柱状图）
3. 数据分析（文字 + Mermaid流程图）
4. 趋势预测（文字 + Nano Banana配图）
5. 投资建议（文字）

**具体要求**：
- 内容创作者：撰写专业、结构清晰的文字内容
- 数据可视化专家：生成HTML交互式图表（柱状图、折线图等）
- 绘画智能体：生成Mermaid流程图和Nano Banana配图

**数据**：
[在此提供您的数据，例如：]
- 2024年Q1-Q4收益率：5.2%, 7.8%, 4.5%, 6.3%
- 资产配置：股票40%，债券30%，货币30%

**风格要求**：
- 专业、商务
- 配色：橙色主题
- 图表清晰易读
```

## 📋 实际应用示例

### 示例 1：基金季度分析报告

```
@内容创作者 @数据可视化专家 @绘画智能体

协作创建"2024年Q4基金市场分析报告"：

**第一部分：市场概况**
- 内容创作者：撰写市场整体情况分析（500字）
- 数据可视化专家：生成月度收益率折线图

**第二部分：投资策略分析**
- 内容创作者：撰写策略分析（300字）
- 绘画智能体：生成Mermaid流程图展示"投资决策流程"

**第三部分：资产配置建议**
- 内容创作者：撰写配置建议（400字）
- 数据可视化专家：生成饼图展示"推荐资产配置比例"
- 绘画智能体：生成Nano Banana配图展示"投资组合概念"

**第四部分：风险提示**
- 内容创作者：撰写风险提示（200字）

**格式要求**：
- 使用Markdown格式
- 图表和图片嵌入在相应章节
- 保持专业报告风格
```

### 示例 2：产品分析报告

```
@内容创作者 创建产品分析报告框架

@数据可视化专家 生成以下图表：
1. 用户增长趋势折线图（过去12个月）

3. 用户满意度柱状图

@绘画智能体 生成：
1. Mermaid流程图：产品迭代流程
2. Nano Banana配图：产品概念图

@内容创作者 整合以上所有内容，创建完整的专业报告
```

## 🎨 图表类型选择指南

### 何时使用 HTML 图表（数据可视化专家）

✅ **适合**：
- 数据对比（柱状图）
- 趋势展示（折线图）
- 占比分析（饼图）
- 多维度数据（雷达图）
- 需要交互的图表

**示例**：
```
@数据可视化专家 生成一个展示各基金类型收益率的柱状图
```

### 何时使用 Mermaid（绘画智能体）

✅ **适合**：
- 流程图
- 架构图
- 时序图
- 甘特图
- 类图

**示例**：
```
@绘画智能体 生成一个Mermaid流程图，展示"基金投资决策流程"
工具：Mermaid
```

### 何时使用 Nano Banana（绘画智能体）

✅ **适合**：
- 概念图
- 配图
- 封面图
- 示意图
- 需要视觉冲击的图片

**示例**：
```
@绘画智能体 生成一张展示"智能投资"概念的配图
工具：Nano Banana
风格：专业、现代、商务
```

## 💡 最佳实践

### 1. 明确指定工具

在调用绘画智能体时，明确指定使用哪个工具：
- `工具：Mermaid` - 生成流程图
- `工具：Nano Banana` - 生成配图
- `工具：Mermaid, Nano Banana` - 同时生成两种

### 2. 提供具体数据

✅ **好的示例**：
```
@数据可视化专家 生成柱状图：
一月：12万，二月：15万，三月：18万
```

❌ **不好的示例**：
```
画个图
```

### 3. 分步骤执行

对于复杂报告，建议分步骤：
1. 先创建文字框架
2. 再生成各个图表
3. 最后整合所有内容

### 4. 使用 Markdown 格式

在整合内容时，要求使用 Markdown 格式，这样：
- 图表代码块可以正确渲染
- 格式清晰易读
- 便于后续编辑

## 🔧 常见问题

### Q1: 如何确保图表和文字完美整合？

**A**: 在最后一步整合时，明确告诉内容创作者：
```
请将以下内容整合，在合适位置插入图表代码块：
- 在"数据分析"章节插入HTML图表
- 在"流程说明"章节插入Mermaid流程图
- 在"概念介绍"章节插入Nano Banana配图
```

### Q2: 如何生成多个图表？

**A**: 可以一次性请求多个图表：
```
@数据可视化专家 生成以下图表：
1. 柱状图：各季度收益
2. 折线图：月度趋势
3. 饼图：资产配置
```

### Q3: 如何确保报告风格统一？

**A**: 在指令中明确要求：
```
要求：
- 使用橙色主题（#FF6B00）
- 专业商务风格
- 图表清晰易读
- 文字简洁专业
```
