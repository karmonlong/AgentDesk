## ğŸš€ Core Functionalities

### 1. Document Processing
Upload documents (PDF, DOCX, TXT, etc.) for automated processing.
*   **Summarize**: Extract core information and generate summaries.
*   **Generate Content**: Create new content based on uploaded documents.
*   **Convert Format**: Convert documents between formats (e.g., Markdown to DOCX).
*   **Extract Table**: Parse and extract tabular data.
*   **Extract Key Points**: Identify and list critical information.
*   **Analyze**: Deep analysis of document content.

### 2. Multi-Agent Chat
Interact with a team of specialized AI agents via the Command Center.
*   **Smart Routing**: The system automatically routes your query to the best-suited agent.
*   **Mentions**: Use `@AgentName` to specifically call an agent (e.g., `@åˆè§„å®˜`).
*   **Context Awareness**: Agents can see uploaded documents and conversation history.

### 3. Visual & Creative Generation
*   **Image Generation**: Create images using models like Nano Banana/Pro.
*   **Diagramming**: Generate diagrams (Flowcharts, Sequence, Gantt, etc.) using Mermaid, PlantUML, or Excalidraw.
*   **Data Visualization**: Create interactive HTML charts (Bar, Line, Pie, etc.) using Chart.js or ECharts.

### 4. Knowledge Base (RAG)
*   **Add to Knowledge Base**: Vectorize and store documents for long-term memory.
*   **Search**: Semantic search across stored documents.
*   **Question Answering**: Agents can retrieve information from the knowledge base to answer questions.

## ğŸ”Œ Key API Endpoints

### File Operations
*   `POST /upload`: Upload and process a file.
*   `GET /download/{filename}`: Download a processed file.
*   `GET /api/files`: List all uploaded files.
*   `DELETE /api/files/{filename}`: Delete a file.

### Chat & Agents
*   `POST /api/chat`: Send a message to the agent system.
*   `GET /api/agents`: Get a list of all available agents.
*   `POST /api/chat/clear`: Clear conversation history.

### Knowledge Base
*   `POST /api/knowledge/add`: Add document to vector store.
*   `POST /api/knowledge/search`: Search the knowledge base.

### Specialized
*   `POST /api/image/generate`: Generate images.
*   `POST /api/draw/generate`: Generate diagrams.
*   `POST /api/prompt/optimize`: Optimize a prompt.
